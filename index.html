¡Comandos!

g=invite • Obten la invitación de Gatoronron
g=help • Información de todos los comandos de GatoronronWize
g=vip • Información de VIP de Gatoronron

g=banid • Banea usuarios de tu servidor (Solo se puede sin razón)
g=apodo • Cambia el apodo de algún usuario (Solo se puede si el usuario tiene menor rango que el del bot)
g=serverinfo • Información del servidor (Puede tener fallos)
g=clear • Elimina una cantidad de mensajes (El máximo es 100)
g=userinfo • Información de algún usuario (Puede tener errores)

$rank • Informate del nivel de tu servidor

g=avatar • Mostrará el avatar de cualquier usuario
g=wasted • Wasted con el perfil de cualquier usuario
